# Manuscript in progress 

## Repository contents

### `Atlantirivulus.xlsx`
Database containing individual-level biological and dietary information used in the analyses, including sampling information, hydrological period, sex, stomach contents, and reproductive measurements.

### `scopus.xlsx`
Database compiled from the systematic literature review on individual dietary specialization in freshwater fishes, including all studies retained after the screening process.

### `code.R`
Main analytical workflow used to process stomach-content data, estimate total niche width (TNW), quantify individual specialization using Bayesian hierarchical models, perform null-model analyses, and test hypotheses.

### `review.R`
Script used to analyze and summarize the literature review dataset, producing figures related to publication trends, habitat types, taxonomic coverage, and seasonal effects reported in previous studies.

## Status
This repository accompanies a manuscript currently in preparation and will be updated throughout the submission, review, and publication process.
